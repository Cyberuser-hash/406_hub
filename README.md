# 406_hub
1) Что бы использовать данную программу необходимо выгрузить дамп sql базы данных(all_databases.sql)На сервере linux-Unix подобной системе
2) Запустить сервер apache2 и заменить /var/www/html на папку html указанную в репозитории
3) Настроить сервер и запустить apache2
При разработке использовались: apache2, mariadb, php, html/css, javascript последних версий
